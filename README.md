# 三相升压独立式逆变器仿真实验

本项目包含了《电力电子系统建模与仿真》课程的实验仿真文件及实验报告源码。

## 📂 项目文件说明

仓库内主要包含以下文件：

* **仿真模型**
    * `*.slx`: MATLAB/Simulink 仿真文件，包含了三相升压独立式逆变器的电路模型与控制逻辑。

* **实验报告**
    * `*.tex`: 实验报告的 LaTeX 源代码。包含了封面、正文及排版设置（已修复下划线对齐问题）。

## 🚀 环境与使用

### 1. 运行仿真
* **软件需求**: MATLAB / Simulink (本模型构建使用 R2024b 完成，R2025 可能导致部分关键库缺失因此不建议使用)
* **步骤**:
    1.  使用 MATLAB 打开 `.slx` 文件。
    2.  等待模型加载完毕后，点击 "Run" 按钮开始仿真。
    3.  通过 Scope 观察输出波形。

### 2. 编译实验报告
* **编译环境**: TeXLive / Overleaf / MacTeX
* **编译方式**: XeLaTeX (推荐，以支持中文)
* **依赖包**: 需要安装 `tabularx`, `graphicx`, `geometry` 等常用宏包。
    * *注：LaTeX 源码已优化 `\fillfield` 命令，解决了封面下划线对齐及空内容报错的问题。*
